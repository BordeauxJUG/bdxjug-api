Votre JUG est partenaire de l'événement


Description de l'évènement

EclipseCon France 2018 sera la cinquième édition de la conférence au cœur de la ville rose. Tous les ans, la conférence rassemble environ 300 personnes autours des thématiques supportées par la fondation Eclipse, thèmes qui ne se limitent plus à l'environnement de développement intégré (EDI/IDE) Eclipse. Les sujets traités sont variés : les objets connectés, Cloud et Devops, Data Science, Modeling, ainsi que Jakarta EE (ex‐Java EE) et Eclipse MicroProfile, sujets qui seront présents en force pour la première fois.

Transfert de Java EE et partenariat avec les JUGs

Suite au transfert récent de Java EE à la fondation Eclipse, des dizaines de projets et des millions de lignes de code sont en train d'être migrés. Glassfish et les JSR dont il dépend sont en cours de transfert pour bientôt devenir Eclipse Glassfish. La fondation souhaite profiter de cette édition 2018 d'EclipseCon en France pour en faire le premier évènement fédérateur de cette nouvelle communauté élargie. Tous les acteurs de l'écosystème Java et membres des différents JUG de France sont invités à participer et contribuer à l'évènement en tant qu'intervenant ou participant. La création du nouveau groupe de travail Jakarta EE, associé à l'investissement dans les autres projets comme Eclipse OpenJ9 ou Eclipse DeeLearning4J, montrent le rôle clé joué aujourd'hui par la fondation dans l'innovation autour de Java.

Le thème EE4J/Java EE et les autres thèmes Java à EclipseCon France 2018

  • Eclipse MicroProfile (microservices pour Java) ;
  • Eclipse OpenJ9 (machine virtuelle Java optimisée) ;
  • Eclipse DeepLearning4J (machine learning) ;
  • Eclipse JNoSQL (NoSQL pour Java EE) ;
  • ou encore Eclipse Vert.x (Reactive Apps pour Java, mais aussi JavaScript, Groovy, Ruby, Ceylon, Scala ou Kotlin).
