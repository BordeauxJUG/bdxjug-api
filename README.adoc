= Bordeaux JUG API

== Description

This project provides a REST API about Bordeaux JUG data.

== Environment Variable

GOOGLE_SHEET_API_KEY:: the api key for https://docs.google.com/spreadsheets/

MEETUP_API_KEY:: the api key for https://api.meetup.com
